Importing a module

Menu path: Modules > Import Module Wizard

This wizard can be found in the "modules" menu of the main screen and can be used to import a module which was exported using the Export Module Wizard. The wizard is powerful yet very simple to use. In the first step you select the module zip file which you would like to import. In the next and final step the import is started.

The following is imported

- The main module and all its related modules.
- Reports that were specifically created for the module.
- The default settings.
- The items belonging to the modules.

Limitation

Data Crow needs to be restarted for the changes to become apparent.
