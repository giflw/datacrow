Altering an existing module

In the main screen of Data Crow you'll find a menu called "modules". This menu contains wizards to customize, create and relate modules. In this chapter we will dive into the alteration of an existing module. This wizard is almost exactly the same as the "Create Module Wizard". The only difference is that not all infomation is allowed to be changed.

Starting the wizard

Start the wizard by selecting "Alter Module Wizard" from the modules menu.

Selecting the module

In the first step of the wizard you have to select the module you want to alter.

Filling the basic information

See the "Create Module Wizard".

Defining fields

See the "Create Module Wizard".

Finishing the Wizard

After all information has been supplied press the "finish" button or the "finish & new" button. The "finish" button finishes the module alteration process. Data Crow needs to restart before the changes can become apparent. The "finish & new" button finishes the module alteration process and restarts the wizard (which can be useful when you want to alter more then one module).
