Altering an existing module

Menu path: Modules > Alter Module Wizard

In the main screen of Data Crow you'll find a menu called "modules". This menu contains wizards to customize, create and relate modules. In this chapter the alteration of an existing module is covered. This wizard is almost exactly the same as the create module wizard. The only difference is that not all information is allowed to be changed.

Starting the wizard

Start the wizard by selecting "Alter Module Wizard" from the modules menu.

Selecting the module

In the first step of the wizard you have to select the module you want to alter.

Filling the basic information

See the "Create Module Wizard".

Defining fields

See the "Defining Fields".

Finishing the Wizard

After all information has been supplied press the "finish" button or the "finish & new" button. The "finish" button finishes the module alteration process. Data Crow needs to restart before the changes can become apparent. The "finish & new" button finishes the module alteration process and restarts the wizard (which can be useful when you want to alter more than one module).
