The tool bar

Item Wizard

Opens the wizard for creating a new item.

New

Opens the form for a new item. The new item can be saved after the information has been filled in.

Open

Opens the selected item.

Save

Saves the selected item.

Save all

Saves all new or modified items.

Delete

Removes a new item (not yet saved) or deletes an existing item.

Filter

Opens the Filter dialog where you can define search conditions.

Re-apply Filter

Re-applies the currently selected filter.

Online Search

Searches one of the online databases such as Amazon, IMDb, and freedb.

Settings

Opens the settings dialog.

Create Report

Opens the reporting dialog.

Import

Opens the import dialog where you can import data from CD or hard disk.

Show Log

Shows system events.

Help

Opens this Help.
