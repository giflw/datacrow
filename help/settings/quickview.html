Quick View Settings

The fields displayed and the order of the fields can be configured using the quick view settings. The quick view settings can be opened from the settings menu.

Enabled

If a field is enabled it is shown in the quick view. Note that a field is only shown when it contains a value.

Direction

Determines where the label is positioned as opposed to the text.

Maximum Length

Sets the maximum display length for the value of this field. Default is 0, meaning the value will be displayed as is.

Order

To move a field select it in the table and use the arrow buttons to move the field up or down.
