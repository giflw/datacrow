Web Field Settings

Here you can define which fields to display in the item overview pages and in which order.

Field

The name of the field for which the settings are applied.

Width

The display width of the field in the item overview pages.

Maximum Text Length

The maximum text length. By default text is not cut off which might cause the text to overflow in the overview pages. Setting a value here will shorten the length of the displayed text.

Visible

Show this field in the overview pages?

Link to details

When set the field becomes a clickable link. The link opens the item detail page.

Quick search

Marking a field as a quick search field will add this field to search area. The search area is displayed in the lower left corner of the web interface.
