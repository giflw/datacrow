Sorting Items

Right-click in item view > Select Sort

Items can be sorted by right clicking on the view and selecting the sort option. The items in the view are sorted on the fields shown in the right panel. The available fields (on which can be sorted) are displayed on the left panel. Fields can be moved by double clicking on them.



In the upper part of the screen you find a text field. Here you can type the name of the field you are looking for.
