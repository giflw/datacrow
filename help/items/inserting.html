New items view


The view in the new items tab shows all new items. Tools such as the File Import and the CD Import can add items to this view. These items are NOT present in the database. To save these items press "save".

Creating a new item

There are several ways you can insert new items:
- Use the item wizard (recommended)
- Open the “File” menu and select “New item”. This will open the item window. You can either type all the values or use the online search function
- Start the Import dialog. more about this
- Search an online database. more about this
- By creating an item based on an existing item. Select an item in the view and select "Edit as New" from the right-click menu.

Using templates

Use templates when you want to set a field to a default value for each newly created item (More about this).
