Mass Update

This tools allows you to update the information stored in Data Crow with the (updated) information from your files (such as mp3's) and/or allows you to use an online service to retrieve missing information. The availability of online search and/or the file re-parsing depend on the selected module. Mass update might even not be available altogether for the selected module.

A combination of file re-parsing and online searching can be used to update all the items shown in the view.
