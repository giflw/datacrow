Auto numbering

Gives an item a unique number. The number can be overwritten by you afterwards. There are several option available for the auto numbering feature.

Enabled

Enables autonumbering for this field. The field will also be enabled and made visible (if it is not already so).

Fill Gaps

Reuses a number. When you have 3 items with a number assigned to each (1,2,3) and you decide to delete the item with number 2 assigned it, you can reuse this number for a new item.
With this option disabled, Data Crow would assign it number 4 instead of filling the gap.

(Re-) number all items

Every item can immediately be given a new and unique number.

Renumber Modus

There are two types of renumbering available; all items can be given a new number or only items with no number are assigned a number.

Order by

You can define the ordering for the renumbering action.
