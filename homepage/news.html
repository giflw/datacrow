News

This section contains news about new releases and other Data Crow related news. The change log text file, which is always updated for every release, can be found here.


02/06/2009 ~ Data Crow 3.4.9 beta has been released

This beta version contains many improvements. The module wizard enables the user to change a field (with limitations). The mass update is now capable to use all search modes of the online search. The tabs as displayed on the item form are now customizable. Beware that this is a beta version. You can find this release in the main download page.


04/03/2009 ~ Data Crow 3.4.8 patch 3 has been released

A couple of important fixes are contained in this patch. This patch enables Mac users to use Data Crow once again. The errors after upgrading to Java 1.6.0 Update 12 have also been fixed.


06/02/2009 ~ Data Crow 3.4.8 patch 2 has been released

This patch fixes the online services for the book module. Changes have been made to the saving mechanism for new items which should fix the NullPointerExceptions some people experienced (unconfirmed fix). The German translation has been updated by Linux (thanks!!).


26/01/2009 ~ Data Crow 3.4.8 patch 1 has been released

Fixes the reporting for the movie module and a possible startup error related to languages.


25/01/2009 ~ Data Crow 3.4.8 has been released

The PDF reporting was broken as well as the module creation wizard. Additionally this release contains improvements for adding images to items and the music album importer.


19/01/2009 ~ New website

The new website is online! This web site has been designed by Bas Uildriks and is a real looker. Thanks Bas!


08/01/2009 ~ Data Crow 3.4.7 has been released

This release fixes the reporting functionality and contains fixes for other minor issues.


02/01/2009 ~ Data Crow 3.4.6 has been released

Due to the overwhelming response and complaints about having to set the DATACROW_HOME directory or to supply the -dir: parameter I have decided to change this with this early release. Oh, and there's something new as well; a French translation! Big thanks to Pierre-Yves DANTEC!


31/12/2008 ~ Data Crow 3.4.5 has been released

This version contains bug fixes for platform security issues. Data Crow did not detect whether the folder and file permissions were set correctly. This could result in loss of data as information could not be written to the database. On startup Data Crow will check these files for their permissions and report back to the user in case of problems.


20/12/2008 ~ Data Crow 3.4.4 has been released

The product has been translated to German (thanks to Klaus Lassleben) and Dutch. Improvements have been made to the resource editor (better UTF8 support and the ability to translate module names and fields). Of course there are again the necessary bug fixes. Please note that a new version of the web module has been made available (1.2). This version contains bug fixes and contains improvements for showing images.
I have invested a lot of my time in documenting the code. This is in preparation of the plugin and online services development documentation.


08/10/2008 ~ Data Crow 3.4.3 has been released

This release contains for the most part bug fixes but also some minor changes.


02/10/2008 ~ Data Crow 3.4.2 has been released

This version of Data Crow improves the online search capabilities and the Data Crow framework as a whole; bugs have been fixed and minor improvements have been made where needed.


28/09/2008 ~ Data Crow 3.4.1 has been released

Data Crow 3.4.1 upgrades the existing loan functionality. A due date has been added and the loan information can now be viewed in a specialized dialog. Relationships to an item are now shown in the item tab. For example: the actor form has a separate tab to show in which movies they have performed. The FreeDB support has been re-added. It was present in older versions and has now been delivered as a separate plugin (with limited support). The tools menu has been split into an administration and a tools menu. The administration menu can be used to maintain genres and the likes and to view the loan information.


08/09/2008 ~ Patch 3.4.0.1 has been released

This patch fixes the loan functionality.


31/08/2008 ~ Data Crow 3.4 has been released

This is the production release of the 3.4 beta version.


26/08/2008 ~ Data Crow 3.4 beta has been released

The new version does contain less new functionality then originally planned. There were a couple of performance issues which needed to be addressed. With this release the startup time, online search and saving of items has been increased dramatically for large databases. The performance for filling the card and table view has also also increased. The upgrade process has been changed. It will now check for the version of Data Crow on startup. The same functionality is used for the backup and restore tool. The location field has been replaced by a container field. These containers are managed from within the container module. This module will be expanded for the next version. Furthermore has new tree become available which shows the file structure of the items. And last but not least a lot of bugs have been fixed.
The change log will tell you what exactly has been fixed and implemented. Please help with the testing of this version!


10/06/2008 ~ Data Crow 3.3 has been released

The web module has been finished! It's available as a separate, optional, package. It needs to be installed on top of a Data Crow 3.3 installation. The web module allows multiple users to connect to one Data Crow instance, remotely, at the same time.
The Data Crow software has been bug fixed, a new online search has been added and improvements were made to the file import.


05/05/2008 ~ Data Crow 3.2.2.1 has been released

This release contains small bug fixes for the tree panel and the quick view layout.


04/05/2008 ~ This website has been updated

The website for this media organiser has been updated.


04/05/2008 ~ Data Crow 3.2.2 has been released

This versions contains the following new features:

  • Improved file import; ability to use local art and added online service selection for the software, book and movie module.
  • Grouping / tree panel has been rewritten to allow deep level grouping and to improve performance.
  • User experience modes; a distinction has been made between beginner and expert users. To the beginning users only the basic option are available. At any time users can switch between experiences modes.



    • This version allows more control when organizing / organising your collections.


    29/04/2008 ~ Data Crow 3.2.1.2 has been released

    It might seem I am moving in the direction of release early and release often.. Well, yes. But I assure you this is just for the time being. I am consolidating the 3.2 version and am anxious to start working on the 3.3 version. However, duty comes first. I want to get this 3.2 version as stable as can be. I think I am almost there, I think most of the bugs have been fixed and I think the next version will not be another pesky 3.2.1.x version but a (glorious) 3.3 version. We'll see!


    25/04/2008 ~ Data Crow 3.2.1.1 has been released

    One of the reasons for this early release is that the Amazon cover search implementation was no longer supported. This service was used by the MusicBrainz online search. The cover search has been re-implemented and is fixed for this version.

    Other fixes focus on the Mass Update and the Migration tool.


    20/04/2008 ~ Data Crow 3.2.1 has been released

    Many fixes and improved sorting.


    30/03/2008 ~ Data Crow 3.2.0.2 has been released

    The 3.2 release of this cataloging product broke the image editing functionality, denied access to certain fields and broke the quick filter functionality. This release resolves these issues. The changes from the 3.2.0.1 patch are also included in this release.


    25/03/2008 ~ Data Crow 3.2.0.1 has been released

    Unfortunately two serious bugs in the 3.2 version made it necessary to release this patch.


    24/03/2008 ~ Data Crow 3.2 has been released

    This latest release introduces a security module, two new online searches and allows for plugin development!


    12/02/2008 ~ Data Crow 3.1.30 has been released

    A new version? So soon? Although I am against releasing version after version within a short span of time here is the brand new 3.1.30 version two weeks after the previous release. The product is slowly recovering from the high impact changes of the 3.1 version. This version again contains crucial fixes. Data Crow is slowly but surely maturing. I have again chosen to create an in between release (this is the 3.1.30 version instead of the 3.2 version) as I want you to benefit as soon as possible from these bug fixes.

    There are also a few changes and new functionality to be found in this version. The File Renaming functionality is the biggest addition and is the first tool which manipulates data outside of Data Crow.


    29/01/2008 ~ Data Crow 3.1.25 has been released

    Unfortunately the 3.1 cataloging software version contained a couple of bugs which could not be fixed in a patch. Beside the necessary bug fixes the 3.1.25 release contains exciting new functionality such as the Drive Manager and two new online book searches.

    Please create a backup before you upgrade to this version! Use the 'backup & restore' tool from your old version.


    21/01/2008 ~ Data Crow 3.1 has been released

    This latest production version is a major overhaul of the internal system of Data Crow. The changes will not be immediately apparent to the average user however it has caused noticeable positive side effects: increased startup performance, lower memory usage, better performing views. This version will allow you to catalog your collection more quickly.
