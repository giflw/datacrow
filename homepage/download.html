Download

The Data Crow cataloguer software is hosted on Sourceforge.net, all files can be downloaded from there. Below you find direct links to the latest releases. Make sure you are downloading the version you want; Beta releases are untested and might thus be unstable. The production releases have been tested and verified to be working correctly. I do encourage people downloading and looking at the beta versions as long as the beta product is run on a backup of the existing database or on a new database.

    • Upgrading

    When upgrading from older versions you will have to follow the upgrade paths:
    Versions older then the 2.12 version should be upgraded to the 2.12 version first.
    Version 2.12 should first be upgraded to the 3.0 version.
    Version 3.0 beta or alpha should first be upgraded to the 3.0 production version.
    Version 3.0 can be upgraded to the latest version.
